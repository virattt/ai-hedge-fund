# 商品适配器快速参考

## 基本使用

### 直接使用CommodityAdapter

```python
from src.markets.commodity import CommodityAdapter

adapter = CommodityAdapter()

# 1. 检查ticker支持
if adapter.supports_ticker("GC=F"):
    print("支持黄金期货")

# 2. 获取价格数据
prices = adapter.get_prices(
    ticker="GC=F",           # 黄金期货
    start_date="2024-01-01",
    end_date="2024-01-31"
)

for price in prices:
    print(f"{price['date']}: ${price['close']:.2f}")

# 3. 获取新闻
news = adapter.get_company_news(
    ticker="GC=F",
    end_date="2024-01-31",
    limit=5
)

for item in news:
    print(f"{item['title']}")
    print(f"  发布: {item['published']}")
    print(f"  链接: {item['link']}")

# 4. 获取财务指标（商品返回空字典）
metrics = adapter.get_financial_metrics("GC=F", "2024-01-31")
# 返回: {}
```

### 通过MarketRouter使用（推荐）

```python
from src.markets.router import MarketRouter

router = MarketRouter()

# 自动识别商品期货并路由
prices = router.get_prices("GC=F", "2024-01-01", "2024-01-31")
news = router.get_company_news("CL=F", "2024-01-31", 5)
metrics = router.get_financial_metrics("SI=F", "2024-01-31")
```

## 支持的期货代码

### 贵金属
- `GC=F` - 黄金 (Gold)
- `SI=F` - 白银 (Silver)
- `PL=F` - 铂金 (Platinum)
- `PA=F` - 钯金 (Palladium)

### 能源
- `CL=F` - 原油 (Crude Oil)
- `NG=F` - 天然气 (Natural Gas)

### 基础金属
- `HG=F` - 铜 (Copper)

### 农产品
- `ZC=F` - 玉米 (Corn)
- `ZW=F` - 小麦 (Wheat)
- `ZS=F` - 大豆 (Soybean)
- `CT=F` - 棉花 (Cotton)
- `KC=F` - 咖啡 (Coffee)
- `SB=F` - 糖 (Sugar)
- `CC=F` - 可可 (Cocoa)

## 数据格式

### 价格数据 (get_prices)
```python
{
    "date": "2024-01-01",      # 日期 (YYYY-MM-DD)
    "open": 1800.50,           # 开盘价
    "high": 1820.00,           # 最高价
    "low": 1795.00,            # 最低价
    "close": 1810.00,          # 收盘价
    "volume": 50000            # 成交量
}
```

### 新闻数据 (get_company_news)
```python
{
    "title": "Gold prices rise...",              # 新闻标题
    "published": "2024-01-01T10:00:00Z",        # 发布时间 (ISO格式)
    "source": "google",                          # 来源
    "link": "http://example.com/news",          # 新闻链接
    "sentiment": None                            # 情绪 (未分析)
}
```

### 财务指标 (get_financial_metrics)
```python
{}  # 商品期货没有财务指标，始终返回空字典
```

## 异常处理

### ValueError - ticker格式不支持
```python
try:
    adapter.get_company_news("AAPL", "2024-01-31", 5)
except ValueError as e:
    print(f"错误: {e}")  # "不支持的ticker格式: AAPL"
```

### Exception - 数据获取失败
```python
try:
    prices = adapter.get_prices("GC=F", "2024-01-01", "2024-01-31")
except Exception as e:
    print(f"获取失败: {e}")  # "获取GC=F价格数据失败: ..."
```

### 新闻获取失败（不抛出异常）
```python
# 新闻获取失败时返回空列表，不会抛出异常
news = adapter.get_company_news("GC=F", "2024-01-31", 5)
if not news:
    print("未获取到新闻")
```

## 集成测试

创建一个测试脚本：

```python
#!/usr/bin/env python3
from src.markets.router import MarketRouter
from datetime import datetime, timedelta

router = MarketRouter()

# 测试多个市场
tickers = ["AAPL", "600000.SH", "0700.HK", "GC=F"]
end_date = datetime.now().strftime("%Y-%m-%d")
start_date = (datetime.now() - timedelta(days=7)).strftime("%Y-%m-%d")

for ticker in tickers:
    print(f"\n测试 {ticker}:")
    try:
        prices = router.get_prices(ticker, start_date, end_date)
        print(f"  价格数据: {len(prices)} 条")

        news = router.get_company_news(ticker, end_date, 3)
        print(f"  新闻数据: {len(news)} 条")

        metrics = router.get_financial_metrics(ticker, end_date)
        print(f"  财务指标: {len(metrics)} 项")
    except Exception as e:
        print(f"  错误: {e}")
```

## 注意事项

1. **API速率限制**
   - yfinance有速率限制，频繁请求可能失败
   - 建议：添加缓存或延时

2. **新闻数据**
   - 使用Google News RSS，可能偶尔不可用
   - 失败时返回空列表，不影响其他功能

3. **交易时间**
   - 期货市场有交易时间限制
   - 非交易日可能获取不到当日数据

4. **数据延迟**
   - 免费API可能有15-20分钟延迟
   - 实时数据需要付费API

## 性能优化建议

### 使用缓存
```python
from functools import lru_cache

@lru_cache(maxsize=100)
def get_cached_prices(ticker, start, end):
    return adapter.get_prices(ticker, start, end)
```

### 批量请求
```python
from concurrent.futures import ThreadPoolExecutor

def fetch_all(tickers):
    with ThreadPoolExecutor(max_workers=5) as executor:
        futures = {
            executor.submit(
                router.get_prices, ticker, start, end
            ): ticker
            for ticker in tickers
        }
        results = {}
        for future in futures:
            ticker = futures[future]
            try:
                results[ticker] = future.result()
            except Exception as e:
                results[ticker] = None
        return results
```

## 故障排除

### 问题1: "Too Many Requests"
**原因**: API速率限制
**解决**: 添加延时或使用缓存

```python
import time
time.sleep(1)  # 请求之间等待1秒
```

### 问题2: 获取不到新闻
**原因**: RSS暂时不可用
**解决**: 检查网络连接，稍后重试

### 问题3: 价格数据为空
**原因**: 非交易日或ticker无效
**解决**: 检查ticker格式和日期

### 1. 核心功能

#### 文件位置
- 实现: `/src/markets/commodity.py`
- 测试: `/tests/markets/test_commodity.py`

#### 方法实现

**1.1 `supports_ticker(ticker: str) -> bool`**
- 识别期货格式：`XX=F`（如 GC=F, CL=F）
- 排除无效格式（如 "=F"）
- 支持的期货代码：
  - GC=F: 黄金期货
  - SI=F: 白银期货
  - CL=F: 原油期货
  - NG=F: 天然气期货
  - HG=F: 铜期货
  - 等等

**1.2 `get_prices(ticker, start_date, end_date) -> List[Dict]`**
- 使用 **yfinance** 库获取期货历史价格
- 返回格式：
  ```python
  {
    "date": "2024-01-01",
    "open": 1800.0,
    "high": 1820.0,
    "low": 1795.0,
    "close": 1810.0,
    "volume": 50000
  }
  ```
- 异常处理：数据获取失败时抛出详细错误信息

**1.3 `get_company_news(ticker, end_date, limit) -> List[Dict]`**
- 商品没有公司新闻，使用 **Google News RSS** 搜索商品名称
- 商品名称映射表（`COMMODITY_NAMES`）:
  - GC → Gold
  - SI → Silver
  - CL → Crude Oil
  - NG → Natural Gas
  - 等14种常见商品
- 返回格式：
  ```python
  {
    "title": "Gold prices rise on inflation fears",
    "published": "2024-01-01T10:00:00Z",
    "source": "google",
    "link": "http://example.com/news",
    "sentiment": None
  }
  ```
- 使用 **feedparser** 库解析RSS
- 异常处理：失败时返回空列表（不阻断主流程）

**1.4 `get_financial_metrics(ticker, end_date) -> Dict`**
- 商品期货没有财务指标
- 返回空字典 `{}`
- 验证ticker格式，不支持时抛出 `ValueError`

**1.5 `_extract_commodity_name(ticker: str) -> str`**
- 辅助方法：从ticker提取商品名称
- 用于新闻搜索
- 支持14种常见商品映射
- 未知代码返回去掉=F的原始代码

## 使用示例

```python
from src.markets.commodity import CommodityAdapter

# 创建适配器
adapter = CommodityAdapter()

# 检查ticker支持
adapter.supports_ticker("GC=F")  # True
adapter.supports_ticker("AAPL")  # False

# 获取黄金期货价格
prices = adapter.get_prices(
    ticker="GC=F",
    start_date="2024-01-01",
    end_date="2024-01-31"
)
# 返回: [{"date": "2024-01-01", "open": 1800.0, ...}, ...]

# 获取黄金相关新闻
news = adapter.get_company_news(
    ticker="GC=F",
    end_date="2024-01-31",
    limit=5
)
# 返回: [{"title": "Gold prices...", "published": "...", ...}, ...]

# 获取财务指标（商品返回空字典）
metrics = adapter.get_financial_metrics(
    ticker="GC=F",
    end_date="2024-01-31"
)
# 返回: {}
```

## 已知限制

1. **yfinance API速率限制**
   - 频繁请求可能遇到 "Too Many Requests" 错误
   - 建议：生产环境使用缓存或付费API

2. **Google News RSS可用性**
   - RSS feed可能偶尔不可用
   - 已实现：失败时返回空列表，不阻断主流程

3. **商品名称映射**
   - 当前支持14种常见商品
   - 未知代码使用原始代码搜索新闻

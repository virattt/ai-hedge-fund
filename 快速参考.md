# AI 对冲基金 - 快速参考

## 🚀 快速开始

### 运行程序（中文输出 + 数据库存储）

```bash
poetry run python src/main.py \
  --tickers AAPL,MSFT \
  --analysts-all \
  --model "MiniMax-M2.5" \
  --start-date 2024-02-28 \
  --end-date 2024-03-01
```

## 📊 数据库配置

### .env 文件
```bash
DB_HOST=localhost
DB_PORT=3306
DB_NAME=hedge-fund
DB_USER=root
DB_PASSWORD=12345678
```

### 创建数据库
```sql
CREATE DATABASE `hedge-fund` CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

## 📝 常用 SQL 查询

### 查看最近的交易
```sql
SELECT * FROM trading_sessions ORDER BY created_at DESC LIMIT 10;
```

### 查看决策详情
```sql
SELECT td.*, ts.model_name
FROM trading_decisions td
JOIN trading_sessions ts ON td.session_id = ts.id
WHERE ts.id = 1;
```

### 查看分析师信号
```sql
SELECT analyst_name, ticker, signal, confidence
FROM analyst_analyses
WHERE session_id = 1
ORDER BY analyst_name;
```
